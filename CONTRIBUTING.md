# Contributing to TCS_TinyClangbasedServer

We welcome contributions! As a project focused on simplicity and performance, we ask that you follow these guidelines:

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Keep changes concise and well-documented.
4. Ensure the server compiles with both Clang and GCC.
5. Submit a Pull Request with a clear description of your changes.

## Areas for Improvement
- Multi-threading support.
- Performance optimizations for file I/O.
- Additional HTTP method support.
- Configurable port and root directory via CLI arguments.

# 기여 방법

기여를 환영합니다! 이 프로젝트는 단순함과 성능에 초점을 맞추고 있으므로 다음 지침을 따라주시기 바랍니다:

## 기여 절차
1. 저장소를 포크합니다.
2. 기능 추가 또는 버그 수정을 위한 새 브랜치를 만듭니다.
3. 변경 사항을 간결하게 유지하고 잘 문서화합니다.
4. Clang과 GCC 모두에서 컴파일되는지 확인합니다.
5. 변경 사항에 대한 명확한 설명과 함께 Pull Request를 제출합니다.

## 개선 가능 영역
- 멀티스레딩 지원.
- 파일 I/O 성능 최적화.
- 추가 HTTP 메서드 지원.
- CLI 인수를 통한 설정 가능한 포트 및 루트 디렉토리.
